# Key Performance Indicators

**Definition**: Key performance indicators (KPIs) are measurable values that demonstrate how effectively an organization is achieving its objectives.

**Examples of KPIs**: 
* Task completion rate: The percentage of tasks completed within a given timeframe.
* Average task duration: The average time taken to complete a task.
* Number of dependencies resolved: The number of dependencies that have been resolved.
* Average team velocity: The average amount of work done by the team in a given timeframe.
